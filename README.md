# green-niri-noctalia

A clean blue-themed rice for Niri and CachyOS. Optimized for performance and aesthetics on laptops and desktop PCs.

📸 Preview



🛠 Prerequisites
To get this exact look, ensure you have the following packages installed:

WM: Niri

Terminal: Alacritty / Kitty

Fonts: Nerd Fonts (Required for icons)

Audio Visualizer: Cava

Matrix: Cmatrix
